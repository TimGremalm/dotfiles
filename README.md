# dotfiles
My configuration files for bash, git, i3, tmux, vim, weechat.

## Vanilla
A more plain installation of tools and config.
### Install
```bash
make vanilla
```
### Uninstall
```bash
make unstow_vanilla
```

## Cozy
A cozier version of vanilla including vim plugins etc.
### Install
```bash
make cozy
```
### Uninstall
```bash
make unstow_cozy
```
